I. Introduction

L'IA prend la date actuelle et les événements réels pour créer une introduction immersive. Vous choisissez :

  • Le pays à diriger (tous les pays du monde)
  • La difficulté : Facile / Normal / Difficile / Réaliste

II. Objectif

Survivre jusqu'à la fin de votre mandat. Vous pouvez briguer un second mandat.

L'IA s'assure que vous n'y parvenez pas facilement : crises, scandales, mouvements sociaux, pressions internationales…

III. Configuration

  • Tous les 24 mois : crise mondiale majeure
  • Durée du mandat adaptée au pays choisi
